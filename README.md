# vitae

This is where I maintain my CV in yaml format to be rendered to DF and HTML using [RenderCV](https://rendercv.com/).

After cloning, run the setup script once to enable the git hooks:

```powershell
.\Setup.ps1
```

Test the changes locally using the `rendercv.ps1` powershell script.

Once pushed to the server the published versions will be generated using github actions.
